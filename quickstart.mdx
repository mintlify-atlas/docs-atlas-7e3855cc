Get the Forsaken Casino memory game running in your browser in just a few steps.

<Steps>
  <Step title="Download the game files">
    Clone or download the Memorama game files to your local machine:
    
    ```bash
    git clone <repository-url>
    cd memorama-webvr
    ```
    
    Or download the ZIP file and extract it to a folder.
  </Step>

  <Step title="Verify your file structure">
    Make sure you have the following files and folders:
    
    ```
    memorama-webvr/
    ├── index.html
    ├── img/
    │   ├── img1.jpg
    │   ├── img2.jpg
    │   ├── img3.jpg
    │   ├── img4.jpg
    │   └── atras.jpeg
    ├── piso.jpg
    ├── lados.jpg
    ├── cartas.mp3
    └── fondo.mp3
    ```
  </Step>

  <Step title="Open in a WebVR-compatible browser">
    Simply open the `index.html` file in your browser:
    
    **Option 1: Double-click**
    - Navigate to the folder and double-click `index.html`
    
    **Option 2: Use a local server (recommended)**
    ```bash
    # Using Python 3
    python -m http.server 8000
    
    # Using Node.js
    npx http-server -p 8000
    ```
    
    Then open `http://localhost:8000` in your browser.
    
    <Note>
      Using a local server is recommended for best performance and to avoid CORS issues with assets.
    </Note>
  </Step>

  <Step title="Start playing">
    The game will load automatically. You'll see:
    - A dark casino environment with red ambient lighting
    - A timer showing "VIDA: 40s" at the top of your view
    - 8 cards arranged on a casino table
    
    **How to play:**
    - Look at a card (hover your red cursor over it) to flip it
    - The timer starts when you flip your first card
    - Match all 4 pairs before time runs out
    - Find all pairs to win and "SALISTE CON VIDA" (escape alive)
    
    <Warning>
      You have 40 seconds to find all matching pairs. If time runs out, you'll see "PERDISTE TU ALMA" (you lost your soul) and the game will restart.
    </Warning>
  </Step>
</Steps>

## Supported Browsers

The game works best on:
- **Chrome** (recommended)
- **Firefox**
- **Edge**
- **Safari** (limited VR support)

<Note>
  For VR headset support, you'll need a WebXR-compatible browser. See the [Setup Guide](/setup) for details.
</Note>
