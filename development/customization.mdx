## Card Customization

### Adding/Changing Card Images

Card images are defined in `<a-assets>` at `index.html:9-14`:

```html
<a-assets>
  <img id="img1" src="img/img1.jpg">
  <img id="img2" src="img/img2.jpg">
  <img id="img3" src="img/img3.jpg">
  <img id="img4" src="img/img4.jpg">
  <img id="back" src="img/atras.jpeg">
</a-assets>
```

<Steps>
  <Step title="Replace image files">
    Replace `img/img1.jpg` through `img/img4.jpg` with your own images
  </Step>
  <Step title="Change card back">
    Replace `img/atras.jpeg` to customize the back texture
  </Step>
  <Step title="Add more pairs">
    Add new `<img>` tags with unique IDs (e.g., `id="img5"`)
  </Step>
  <Step title="Update cardData array">
    Modify the array at `index.html:65` to include new pairs
  </Step>
</Steps>

#### Example: 6 Pairs (12 Cards)

```html
<!-- In <a-assets> -->
<img id="img5" src="img/img5.jpg">
<img id="img6" src="img/img6.jpg">
```

```javascript
// At index.html:65
const cardData = [
    'img1', 'img1', 
    'img2', 'img2', 
    'img3', 'img3', 
    'img4', 'img4',
    'img5', 'img5',  // New pair
    'img6', 'img6'   // New pair
];
```

<Warning>
  You must also update the grid layout logic at `index.html:90-91` to accommodate more cards (e.g., 3x4 grid instead of 4x2).
</Warning>

### Changing Card Dimensions

Card size is set at `index.html:94-96`:

```javascript
card.setAttribute('width', '0.35'); 
card.setAttribute('height', '0.5'); 
card.setAttribute('depth', '0.01');
```

<CodeGroup>
```javascript Larger Cards
card.setAttribute('width', '0.5');   // 43% wider
card.setAttribute('height', '0.7');  // 40% taller
card.setAttribute('depth', '0.02');  // Thicker
```

```javascript Smaller Cards
card.setAttribute('width', '0.25');  // 29% smaller
card.setAttribute('height', '0.35'); // 30% shorter
card.setAttribute('depth', '0.005'); // Thinner
```
</CodeGroup>

<Note>
  Remember to adjust spacing in the grid layout (`index.html:90-91`) if you change card dimensions.
</Note>

### Grid Layout Customization

Current 4x2 grid defined at `index.html:90-91`:

```javascript
const x = (index % 4) * 0.4;           // 4 columns, 0.4 spacing
const y = Math.floor(index / 4) * -0.6; // 2 rows, 0.6 spacing
```

#### Example: 3x4 Grid (12 Cards)

```javascript
const x = (index % 3) * 0.5;           // 3 columns, 0.5 spacing
const y = Math.floor(index / 3) * -0.7; // 4 rows, 0.7 spacing
```

#### Example: 5x2 Grid (10 Cards)

```javascript
const x = (index % 5) * 0.35;          // 5 columns, 0.35 spacing
const y = Math.floor(index / 5) * -0.6; // 2 rows, 0.6 spacing
```

### Number of Pairs

Change win condition at `index.html:138`:

```javascript
if (pairsFound === 4) win(); // 4 pairs = 8 cards
```

For 6 pairs (12 cards):

```javascript
if (pairsFound === 6) win(); // 6 pairs = 12 cards
```

  Must match the number of unique pairs in `cardData` array at `index.html:65`.
</Warning>

## Audio Customization

### Background Music

Defined at `index.html:18`:

```html
<audio id="background-music" src="fondo.mp3" preload="auto"></audio>
```

And played at `index.html:70`:

```javascript
document.querySelector('#bgMusic').components.sound.playSound();
```

<Steps>
  <Step title="Replace audio file">
    Replace `fondo.mp3` with your own music loop
  </Step>
  <Step title="Adjust volume">
    Modify `volume` at `index.html:21` (default: 0.3)
  </Step>
</Steps>

```html
<a-entity id="bgMusic" sound="src: #background-music; loop: true; volume: 0.5">
```

### Card Flip Sound

Defined at `index.html:17`:

```html
<audio id="click-sound" src="cartas.mp3"></audio>
```

Played at `index.html:108`:

```javascript
new Audio(document.querySelector('#click-sound').src).play();
```

Replace `cartas.mp3` with your preferred sound effect.

<Note>
  New Audio instances are created each time to allow overlapping sounds when flipping multiple cards quickly.
</Note>


<CodeGroup>
```html Horror Red (Default)

```



```html No Fog
<a-scene>
<!-- Remove fog attribute entirely -->
```
</CodeGroup>

#### Lighting Color

Flickering light at `index.html:33`:

```html
<a-light id="flicker" type="point" intensity="1" position="0 3 0" color="#f00"
         animation="property: intensity; from: 0.2; to: 1.2; dur: 150; 
                    loop: true; dir: alternate">
```

Change `color="#f00"` to any hex color:
- `#00f` - Blue
- `#0f0` - Green
- `#ff0` - Yellow
- `#fff` - White

## Animation Speeds

All animations can be tuned at these locations:

| Animation | Location | Default | Parameter |
|-----------|----------|---------|----------|
| Card flip | `index.html:122` | 250ms | `dur: 250` |
| Card shrink | `index.html:133` | 300ms | `dur: 300` |
| Light flicker | `index.html:34` | 150ms | `dur: 150` |
| Death fade | `index.html:82` | 1000ms | `dur: 1000` |

<Tip>
  Lower `dur` values make animations faster and more intense. Higher values create slower, dramatic effects.
</Tip>

## Camera Settings

Camera rig at `index.html:23-24`:

```html
<a-entity id="rig" position="0 1.6 1.2">
  <a-entity id="camera" camera look-controls 
            wasd-controls="acceleration: 10" rotation="-25 0 0">
```

<AccordionGroup>
  <Accordion title="Camera Height">
    Modify `position` Y value:
    ```html
    position="0 2.0 1.2"  <!-- Taller player -->
    position="0 1.3 1.2"  <!-- Shorter player -->
    ```
  </Accordion>
  
  <Accordion title="Camera Distance">
    Modify `position` Z value:
    ```html
    position="0 1.6 2.0"  <!-- Farther from table -->
    position="0 1.6 0.5"  <!-- Closer to table -->
    ```
  </Accordion>
  
  <Accordion title="Camera Angle">
    Modify `rotation` X value:
    ```html
    rotation="-35 0 0"  <!-- Look down more -->
    rotation="-15 0 0"  <!-- Look down less -->
    rotation="0 0 0"    <!-- Straight ahead -->
    ```
  </Accordion>
  
  <Accordion title="Movement Speed">
    Modify `wasd-controls` acceleration:
    ```html
    wasd-controls="acceleration: 20"  <!-- Faster -->
    wasd-controls="acceleration: 5"   <!-- Slower -->
    ```
  </Accordion>
</AccordionGroup>

<Note>
  All customizations take effect immediately after saving `index.html` and refreshing the browser.
</Note>
